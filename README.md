# 📱 Direktori Kontak menggunakan Pencarian Cepat
Kelompok 3

Anggota : Aga, Athallah, Gabriel, Gibran, Nabil

---
## 📝 Deskripsi Proyek

**Aplikasi Kontak berbasis Binary Search Tree.** Kontak disimpan secara terurut berdasarkan nama, mendukung pencarian cepat, penambahan, dan penghapusan kontak.

---
## ⚙️ Fitur yang harus Diimplementasikan
<ul>
  <li>Insert, delete, search kontak</li>
  <li>Traversal in-order (urutan alfabetis otomatis)</li>
  <li>Tampilkan kontak dalam rentang nama tertentu</li>
  <li>Analisis keseimbangan pohon</li>
</ul>
